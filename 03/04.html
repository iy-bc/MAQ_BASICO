Los eventos son acciones o sucesos que ocurren en la página web, como por ejemplo hacer clic en un botón, mover el mouse sobre un elemento, o enviar un formulario. Los eventos permiten que el usuario interactúe con la página web y que ésta responda a las acciones del usuario.

JavaScript proporciona una manera de manejar eventos en la página web mediante funciones que se ejecutan cuando ocurre un evento específico. Algunos de los eventos más comunes son:

  • click: se activa cuando el usuario hace clic en un elemento.
  • hover: se activa cuando el usuario pasa el ratón por encima de un elemento.
  • submit: se activa cuando se envía un formulario.
  • keyup: se activa cuando el usuario suelta una tecla después de presionarla.
  • load: se activa cuando la página web se ha cargado completamente.
  • resize: se activa cuando el usuario cambia el tamaño de la ventana del navegador.

Para agregar un evento a un elemento HTML, se utiliza la propiedad "on" seguida del nombre del evento. Por ejemplo, para agregar un evento de clic a un botón, se puede usar el siguiente código:

 
<button id="miBoton">Haz clic aquí</button> <script> document.getElementById("miBoton").onclick = function() { alert("¡Hola mundo!"); }; </script>

En este ejemplo, se ha utilizado la función getElementById para seleccionar el botón con el id "miBoton", y se ha agregado un evento de clic utilizando la propiedad onclick. La función que se ejecutará cuando se haga clic en el botón es una función anónima que muestra una ventana de alerta con el mensaje "¡Hola mundo!".

También es posible agregar eventos utilizando el método addEventListener, que permite agregar múltiples eventos a un elemento. Por ejemplo, para agregar eventos de clic y hover a un botón, se puede utilizar el siguiente código:

 
<button id="miBoton">Haz clic aquí</button> <script> var boton = document.getElementById("miBoton"); boton.addEventListener("click", function() { alert("¡Hola mundo!"); }); boton.addEventListener("mouseover", function() { boton.style.backgroundColor = "red"; }); boton.addEventListener("mouseout", function() { boton.style.backgroundColor = ""; }); </script>

En este ejemplo, se ha utilizado el método addEventListener para agregar tres eventos al botón: un evento de clic, un evento de pasar el ratón por encima (mouseover), y un evento de quitar el ratón (mouseout). La función que se ejecuta cuando se hace clic en el botón muestra una ventana de alerta con el mensaje "¡Hola mundo!". Las funciones que se ejecutan cuando se pasa el ratón por encima y se quita el ratón cambian el color de fondo del botón a rojo y lo restauran a su valor original, respectivamente.
