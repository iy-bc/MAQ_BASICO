El DOM (Document Object Model) es una representación de la estructura de un documento HTML o XML como un árbol de objetos, donde cada objeto representa una parte del documento, como un elemento, un atributo o un fragmento de texto. El DOM permite interactuar con la página web mediante JavaScript, permitiendo la selección, modificación y eliminación de elementos y atributos.

La selección de elementos en el DOM se realiza mediante los métodos proporcionados por el objeto document. Los métodos más comunes son:

  • getElementById(): selecciona un elemento por su ID único.
  • getElementsByClassName(): selecciona todos los elementos que tienen una clase específica.
  • getElementsByTagName(): selecciona todos los elementos con una etiqueta HTML específica.
  • querySelector(): selecciona el primer elemento que coincide con un selector CSS específico.
  • querySelectorAll(): selecciona todos los elementos que coinciden con un selector CSS específico.

Estos métodos devuelven un objeto que representa el elemento seleccionado, y se pueden encadenar para seleccionar elementos anidados o filtrar los resultados. Por ejemplo, para seleccionar todos los elementos p con la clase "ejemplo", se puede usar el siguiente código:

 
const elementos = document.querySelectorAll("p.ejemplo");

Esto devuelve una lista de objetos que representan los elementos seleccionados, que se pueden recorrer y manipular mediante JavaScript.

Recorriendo el DOM

Existen varias formas de recorrer el DOM en JavaScript, algunas de ellas son:

  1. Recorriendo los nodos hermanos:

Podemos recorrer los nodos hermanos de un elemento a través de las propiedades previousSibling y nextSibling para navegar hacia arriba y hacia abajo en el árbol del DOM respectivamente. Por ejemplo:

 
let primerElemento = document.getElementById("primero"); let siguienteElemento = primerElemento.nextSibling; let anteriorElemento = primerElemento.previousSibling;
  1. Recorriendo los nodos hijos:

Podemos recorrer los nodos hijos de un elemento a través de la propiedad childNodes. Por ejemplo:

 
let padre = document.getElementById("padre"); let hijos = padre.childNodes; for (let i = 0; i < hijos.length; i++) { console.log(hijos[i]); }

También podemos recorrer los nodos hijos de un elemento a través de las propiedades firstChild y lastChild para acceder al primer y último hijo respectivamente. Por ejemplo:

 
let primerHijo = padre.firstChild; let ultimoHijo = padre.lastChild;
  1. Recorriendo los nodos padres:

Podemos recorrer los nodos padres de un elemento a través de la propiedad parentNode. Por ejemplo:

 
let elementoHijo = document.getElementById("hijo"); let elementoPadre = elementoHijo.parentNode;
  1. Encontrar coincidencias:

Podemos encontrar nodos que coinciden con ciertos criterios utilizando los métodos querySelector y querySelectorAll. querySelector nos devuelve el primer elemento que cumple el criterio especificado, mientras que querySelectorAll nos devuelve todos los elementos que cumplen el criterio especificado. Por ejemplo:

 
let elemento = document.querySelector("#mi-elemento"); let elementos = document.querySelectorAll(".clase-elemento");

Además, podemos buscar elementos por su etiqueta utilizando el método getElementsByTagName. Por ejemplo:

 
let elementosP = document.getElementsByTagName("p");

Estas son solo algunas de las formas en que podemos recorrer y seleccionar elementos en el DOM de una página web utilizando JavaScript.
