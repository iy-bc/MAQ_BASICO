JavaScript es un lenguaje de programación que se utiliza para crear sitios web interactivos y aplicaciones web. Fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape Communications Corporation.

Algunos conceptos básicos de programación que se aplican a JavaScript son:

  • Variables: Las variables son contenedores para almacenar valores. En JavaScript, se pueden declarar variables usando la palabra clave "var", "let" o "const". Por ejemplo:
 
var x = 5;
let y = "Hola";
const z = true;
  • Operadores: Los operadores son símbolos o palabras reservadas que se utilizan para realizar operaciones. Algunos operadores comunes en JavaScript son el operador de asignación "=", el operador aritmético "+", el operador de comparación "==" y el operador lógico "&&". Por ejemplo:
 
var a = 5 + 3; // a será igual a 8
var b = "Hola" + " Mundo"; // b será igual a "Hola Mundo"
var c = (5 == 5) && (3 < 5); // c será igual a true

  • Condicionales: Los condicionales se utilizan para tomar decisiones basadas en una condición. En JavaScript, se pueden utilizar los condicionales "if", "else if" y "else". Por ejemplo:
 
var edad = 18;
if (edad >= 18)
{ console.log("Eres mayor de edad");
} else {
console.log("Eres menor de edad");
}

En este ejemplo, si la variable "edad" es mayor o igual a 18, se imprimirá en la consola el mensaje "Eres mayor de edad", de lo contrario, se imprimirá el mensaje "Eres menor de edad".

Estos son solo algunos conceptos básicos de programación que se aplican a JavaScript. Hay muchos otros conceptos y elementos de lenguaje que se pueden utilizar para crear aplicaciones más complejas.

Estructuras de control

Estructura de control Sintaxis Explicación
if if (condición) { // código si la condición es verdadera } Evalúa una condición y ejecuta el código dentro del bloque de llaves si es verdadera.
if...else if (condición) { // código si la condición es verdadera } else { // código si la condición es falsa } Evalúa una condición y ejecuta el primer bloque de código si es verdadera y el segundo bloque si es falsa.
if...else if...else if (condición1) { // código si la condición 1 es verdadera } else if (condición2) { // código si la condición 2 es verdadera } else { // código si todas las condiciones son falsas } Evalúa varias condiciones en orden y ejecuta el bloque de código correspondiente al primer resultado verdadero. Si ninguna condición es verdadera, se ejecuta el bloque de código final.
switch switch (expresión) { case valor1: // código si la expresión coincide con valor1 break; case valor2: // código si la expresión coincide con valor2 break; default: // código si la expresión no coincide con ningún valor } Evalúa una expresión y ejecuta el bloque de código correspondiente al primer caso que coincida. Si no hay coincidencias, se ejecuta el bloque de código "default". El break es necesario para evitar que se ejecuten todos los casos posteriores.
for for (inicialización; condición; actualización) { // código a repetir } Crea un bucle que se ejecuta mientras la condición es verdadera. La inicialización se ejecuta una vez antes del bucle, la actualización se ejecuta después de cada iteración.
while while (condición) { // código a repetir } Crea un bucle que se ejecuta mientras la condición es verdadera.
do...while do { // código a repetir } while (condición); Crea un bucle que se ejecuta al menos una vez y luego mientras la condición es verdadera.
for...in for (variable in objeto) { // código a repetir } Itera sobre todas las propiedades enumerables de un objeto. La variable toma el valor de cada propiedad en cada iteración.
for...of for (variable of iterable) { // código a repetir } Itera sobre cada elemento de un objeto iterable, como un array. La variable toma el valor de cada elemento en cada iteración.
