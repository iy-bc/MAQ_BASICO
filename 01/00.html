Fundamentos en HTML

En esta lección aprenderás los fundamentos del lenguaje de hojas de estilo en cascada (CSS). Aprenderás a aplicar estilos visuales y de diseño a tus documentos HTML utilizando selectores, propiedades y valores de CSS. También aprenderás cómo usar el modelo de caja y el sistema de posicionamiento para controlar la ubicación y el tamaño de los elementos en una página web. Además, comprenderás cómo utilizar los medios de consulta (media queries) para crear diseños responsivos que se adapten a diferentes tamaños de pantalla y dispositivos.

Estructura básica

La estructura básica de un documento HTML5 es esencial para cualquier página web.

Etiquetas HTML

Uso de las etiquetas para diseñar y estructurar una página web para lograr el resultado deseado.

Código semántico

El código web semántico es la práctica de utilizar etiquetas HTML que describen el significado y el propósito.

Atributos

Los atributos son valores agregados a las etiquetas HTML para proporcionar información adicional.

Formularios

Los formularios son elementos clave para permitir que los usuarios interactúen con un sitio web o aplicación web.

Multimedia

HTML5 incluye etiquetas multimedia que permiten a los desarrolladores web integrar fácilmente elementos multimedia.
