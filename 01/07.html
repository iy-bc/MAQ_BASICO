HTML5 es la última versión del lenguaje de marcado de hipertexto utilizado para crear y diseñar sitios web. Utiliza etiquetas para estructurar el contenido y proporciona una sintaxis semántica clara para describir el significado del contenido.

Algunas de las etiquetas más comunes son:

  • <html>: define el comienzo y el final del documento HTML.
  • <head>: contiene información sobre el documento, como metadatos, scripts y estilos.
  • <body>: contiene el contenido visible del documento, como texto, imágenes y enlaces.
  • <header>: representa una sección de encabezado o título en una página web.
  • <nav>: define una sección de navegación para enlaces a otras páginas.
  • <main>: define el contenido principal de una página.
  • <section>: define una sección de contenido temático en una página.
  • <article>: define un artículo independiente en una página.
  • <aside>: define un contenido adicional o complementario en una página.
  • <footer>: representa una sección de pie de página en una página web.

El codificado semántico se refiere a utilizar etiquetas HTML apropiadas para el contenido que estás describiendo. Por ejemplo, utilizar <h1> para los títulos de las secciones, <p> para los párrafos y <ul> y <ol> para las listas.

Los elementos de bloque son aquellos que ocupan toda la anchura disponible y que comienzan en una nueva línea, mientras que los elementos en línea son aquellos que no comienzan en una nueva línea y que se pueden colocar junto a otros elementos.

Algunos de los atributos más comunes son:

  • class: se utiliza para definir una clase CSS para un elemento.
  • id: se utiliza para definir un identificador único para un elemento.
  • href: se utiliza para especificar una URL a la que se debe enlazar.
  • src: se utiliza para especificar la ubicación de un archivo de imagen o de otro tipo.
  • alt: se utiliza para proporcionar un texto alternativo para una imagen.
  • title: se utiliza para proporcionar información adicional sobre un elemento cuando se pasa el cursor sobre él.
  • style: se utiliza para aplicar estilos CSS a un elemento.

En resumen, HTML5 proporciona una sintaxis semántica clara para estructurar el contenido de una página web utilizando etiquetas, y ofrece una variedad de atributos para personalizar y mejorar la presentación del contenido.
