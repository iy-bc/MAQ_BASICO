Además de las etiquetas mencionadas anteriormente, HTML5 también incluye etiquetas multimedia que permiten a los desarrolladores web integrar fácilmente elementos multimedia en sus páginas web. Algunas de estas etiquetas son:

  • audio: se utiliza para insertar archivos de audio en una página web.
  • video: se utiliza para insertar archivos de video en una página web.

Dentro de cada etiquta de multimeda hay estos atributos:

  • source: se utiliza dentro de las etiquetas audio y video para especificar la ubicación del archivo multimedia y el formato del archivo.
  • track: se utiliza dentro de la etiqueta video para agregar subtítulos a un video.

Otras etiquetas dignas de mención para la maquetación básica incluyen:

  • br: se utiliza para crear saltos de línea en una página web.
  • hr: se utiliza para crear una línea horizontal en una página web.
  • script: se utiliza para vincular una un archivo de recursos de script de javascript relacionado.
