• Encabezados (h1-h6): Los encabezados se utilizan para estructurar el contenido de la página en secciones. Cada encabezado tiene un nivel de importancia diferente, donde el encabezado h1 es el más importante y el h6 es el menos importante. Los encabezados ayudan a los motores de búsqueda a entender la estructura del contenido de la página, lo que es importante para mejorar el SEO y la accesibilidad de la página.

  • Párrafos (p): Los párrafos se utilizan para separar el contenido en bloques de texto. Esto facilita la lectura y comprensión del contenido. Los párrafos también pueden ser útiles para aplicar estilos y formatos específicos a bloques de texto dentro de la página.

  • Listas (ul, ol): Las listas se utilizan para presentar información de forma organizada y jerárquica. Las listas pueden ser ordenadas (ol) o no ordenadas (ul) y se pueden utilizar para mostrar elementos como ítems de menú, pasos de un proceso o una serie de instrucciones. Las listas también son útiles para aplicar estilos específicos a los elementos de la lista.

  • Imágenes (img): Las imágenes se utilizan para agregar elementos visuales al contenido de la página. Las imágenes pueden ayudar a explicar el contenido o simplemente añadir interés visual. Las etiquetas de imagen también son útiles para el SEO, ya que permiten incluir descripciones de las imágenes que pueden ser indexadas por los motores de búsqueda.

  • Enlaces (a): Los enlaces se utilizan para conectar diferentes páginas o secciones de una misma página. Los enlaces pueden ser utilizados para navegar entre diferentes secciones de la página, para enlazar a otras páginas o para agregar interactividad a la página. Los enlaces también son importantes para el SEO, ya que permiten a los motores de búsqueda rastrear e indexar diferentes páginas de la web.

Especificaciones de las etiquetas

  1. Encabezados: Los encabezados se usan para indicar la jerarquía del contenido de la página. HTML5 ofrece seis niveles de encabezados, del <h1> al <h6>. Por ejemplo:
 
<h1>Este es un encabezado de nivel 1</h1> <h2>Este es un encabezado de nivel 2</h2>
  1. Párrafos: Los párrafos se usan para separar el contenido en bloques de texto. La etiqueta <p> se utiliza para crear párrafos. Por ejemplo:
 
<p>Este es un párrafo de ejemplo.</p>
  1. Listas: Las listas se utilizan para presentar información en forma de lista. HTML5 ofrece dos tipos de listas: las listas ordenadas (usando la etiqueta <ol>) y las listas no ordenadas (usando la etiqueta <ul>). Por ejemplo:
 
<ul> <li>Elemento 1</li> <li>Elemento 2</li> <li>Elemento 3</li> </ul> <ol> <li>Primer elemento</li> <li>Segundo elemento</li> <li>Tercer elemento</li> </ol>
  1. Imágenes: Las imágenes se utilizan para mostrar gráficos y otros elementos visuales en una página web. La etiqueta <img> se utiliza para insertar imágenes en una página. Por ejemplo:
 
<img src="ruta-de-la-imagen.jpg" alt="Descripción de la imagen">
  1. Enlaces: Los enlaces se utilizan para conectar diferentes páginas web. La etiqueta <a> se utiliza para crear enlaces que van dentro del atributo HREF, si se le pone un atributo TITLE, aparecerá ese texto cuando se pase el ratón por encima. Por ejemplo:
 
<a href="http://www.ejemplo.com">Este es un enlace a un sitio web</a>

Elementos de bloque y en línea

Además, es importante destacar la diferencia entre elementos de bloque y elementos en línea en HTML5. Los elementos de bloque se utilizan para crear bloques de contenido que ocupan todo el ancho disponible y crean un salto de línea después de ellos. Los elementos en línea se utilizan para crear contenido que fluye dentro del texto y no crea un salto de línea después de ellos.

Los elementos de bloque más comunes son:

  • <div>
  • <h1> a <h6>
  • <p>
  • <ul> y <ol>
  • <li>
  • <header>
  • <footer>
  • <nav>
  • <article>
  • <section>
  • <aside>

Los elementos en línea más comunes son:

  • <a>
  • <span>
  • <img>
  • <strong>
  • <em>
  • <br>
  • <input>
  • <button>
  • <label>

Es importante tener en cuenta estas diferencias al diseñar y estructurar una página web para lograr el resultado deseado.
