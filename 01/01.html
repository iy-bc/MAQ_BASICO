La estructura básica de un documento HTML5 es esencial para cualquier página web. Aquí te presento los elementos que la conforman:

  1. DTD (Document Type Definition): Este elemento se ubica en la primera línea del documento HTML5 y especifica la versión del HTML que se está utilizando. En HTML5, se utiliza la siguiente declaración: <!DOCTYPE html>.

  2. Elemento HTML: Es el elemento raíz del documento HTML5 y es el contenedor principal para todo el contenido de la página. Se indica mediante las etiquetas de apertura <html> y de cierre </html>.

  3. Elemento head: El elemento head contiene información meta sobre el documento, como el título de la página (que aparece en la pestaña del navegador), las etiquetas meta para SEO y las referencias a archivos CSS y JavaScript. Se indica mediante las etiquetas de apertura <head> y de cierre </head>.

  4. Elemento body: El elemento body es el contenedor de todo el contenido visible de la página, como texto, imágenes y elementos multimedia. Se indica mediante las etiquetas de apertura <body> y de cierre </body>.

Aquí un ejemplo de estructura básica de un documento HTML5:

<!DOCTYPE html>
<html>
 <head>
 <title>Título de la página</title>
 <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <link rel="stylesheet" href="styles.css">
 <script src="script.js"></script>
 </head>
 <body>
 <h1>Encabezado de nivel 1</h1>
 <p>Este es un párrafo de ejemplo.</p>
 <img src="imagen.jpg" alt="Descripción de la imagen">
 </body>
</html>

Es importante notar que todos los elementos mencionados deben estar presentes en un documento HTML5 válido. A partir de aquí, se pueden agregar más elementos para crear una página web más completa y compleja. 

Debe contener como mínimo una estructura básica: un elemento DOCTYPE que identifique que es un sitio HTML, una etiqueta HEAD que incluya la codificación de la página, como sería la <meta charset="UTF-8">y una etiqueta de BODY. Dentro de la mísma es donde agregaremos los elementos que conformen la web.
