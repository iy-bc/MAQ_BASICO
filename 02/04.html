El box model es un concepto fundamental en CSS que se refiere a cómo se calcula el tamaño total de un elemento HTML. Cada elemento HTML se representa como un rectángulo en la página, y el box model define cómo se calcula el ancho y alto total de ese rectángulo.

El box model consta de cuatro partes principales: el contenido, el padding, el borde y el margen. Cada una de estas partes afecta al tamaño total del elemento:

  • Contenido: es el área que contiene el contenido real del elemento, como el texto, las imágenes o los vídeos.
  • Padding: es un espacio transparente que rodea el contenido y se encuentra entre el contenido y el borde. El padding se puede ajustar para aumentar o disminuir la distancia entre el contenido y el borde.
  • Borde: es una línea sólida o discontinua que rodea el padding y el contenido del elemento. El borde se puede ajustar en términos de tamaño, estilo y color.
  • Margen: es un espacio transparente que se encuentra fuera del borde y que separa el elemento de otros elementos de la página. El margen se puede ajustar para aumentar o disminuir la distancia entre el elemento y otros elementos de la página.

Para controlar el diseño y tamaño de los elementos HTML5, se pueden utilizar las propiedades de CSS relacionadas con el box model:

  • width: define el ancho del contenido del elemento.
  • height: define la altura del contenido del elemento.
  • padding: define la cantidad de espacio entre el contenido y el borde.
  • border: define el tamaño, estilo y color del borde del elemento.
  • margin: define la cantidad de espacio entre el borde del elemento y otros elementos de la página.

Estas propiedades se pueden aplicar a través de selectores de etiqueta, clase o ID en un archivo CSS externo o en línea en la sección style del archivo HTML.

Por ejemplo, para crear un cuadrado con un borde de 2 píxeles, un padding de 10 píxeles y un margen de 20 píxeles, se podría usar el siguiente código CSS:

 
.square { width: 100px; height: 100px; padding: 10px; border: 2px solid black; margin: 20px; }

Este código crea un elemento con la clase "square" que tiene un ancho y una altura de 100 píxeles, un padding de 10 píxeles, un borde sólido de 2 píxeles y un margen de 20 píxeles. El tamaño total del elemento en la página sería de 144 píxeles de ancho y 144 píxeles de alto.

Modo de disposición de elementos: uso del display

La propiedad CSS "display" define cómo se muestra un elemento HTML. Los valores de "display" pueden ser: block, inline, inline-block, none, flex, grid, table, entre otros.

Aquí hay una tabla con algunos de los valores de "display" más comunes y sus explicaciones:

Valor de "display" Explicación
block El elemento es un bloque, y ocupa todo el ancho disponible en su contenedor padre, y comienza en una nueva línea.
inline El elemento es una línea, y solo ocupa el ancho que necesita. No se puede establecer ancho, altura, margen superior o inferior.
inline-block El elemento es una línea que también acepta ancho, altura, margen superior e inferior, y puede estar en la misma línea.
none El elemento no se muestra.
flex El elemento se comporta como un contenedor flexible.
grid El elemento se comporta como un contenedor de cuadrícula.
table El elemento se comporta como una tabla.
list-item El elemento se comporta como un elemento de lista.

La propiedad "display" es una herramienta poderosa para controlar el diseño y la estructura de una página web, y su correcto uso puede mejorar significativamente la legibilidad y la usabilidad del sitio.

 

Flexbox

Flexbox es una técnica de diseño que permite controlar la disposición y el tamaño de los elementos en un contenedor flexible. Los elementos flexibles se pueden organizar en una sola fila o en una sola columna, o pueden envolverse en múltiples filas o columnas según sea necesario.

La propiedad principal de flexbox es display: flex, que se aplica al contenedor que se va a convertir en un contenedor flexible.

A continuación, se muestra una tabla de las propiedades principales de flexbox y sus ejemplos de uso:

Explicación/Lo que se consigue Propiedad Ejemplo
Indica que el contenedor es un contenedor flexible display display: flex;
Define cómo se distribuyen los elementos a lo largo del eje principal (eje de fila por defecto) justify-content justify-content: center;
Define cómo se distribuyen los elementos a lo largo del eje secundario (eje de columna por defecto) align-items align-items: center;
Define cómo se distribuyen los elementos en un eje secundario adicional cuando hay más de una fila o columna align-content align-content: space-between;
Define cómo se organizan los elementos dentro del contenedor flexible flex-direction flex-direction: row;
Define cómo se ajustan los elementos en el contenedor flexible cuando hay espacio adicional disponible flex-wrap flex-wrap: wrap;
Define cómo se ajusta el tamaño de los elementos flexibles dentro del contenedor flexible flex flex: 1;
Define el tamaño mínimo, preferido y máximo del elemento flexible flex-basis flex-basis: 300px;
Define el orden en el que se deben mostrar los elementos dentro del contenedor flexible order order: 2;

Con flexbox, se pueden lograr varios diseños complejos y dinámicos de manera fácil y eficiente.

Ejemplos de flexbox

  1. Ejemplo de disposición horizontal de elementos en línea utilizando flexbox:
 
<div class="container"> <div class="item">Elemento 1</div> <div class="item">Elemento 2</div> <div class="item">Elemento 3</div> </div>
 
.container { display: flex; } .item { margin: 10px; padding: 10px; background-color: #ddd; }
  1. Ejemplo de disposición vertical de elementos utilizando flexbox:
 
<div class="container"> <div class="item">Elemento 1</div> <div class="item">Elemento 2</div> <div class="item">Elemento 3</div> </div>
 
.container { display: flex; flex-direction: column; } .item { margin: 10px; padding: 10px; background-color: #ddd; }
  1. Ejemplo de centrado de elementos en el centro utilizando flexbox:
 
<div class="container"> <div class="item">Elemento 1</div> <div class="item">Elemento 2</div> <div class="item">Elemento 3</div> </div>
 
.container { display: flex; justify-content: center; align-items: center; } .item { margin: 10px; padding: 10px; background-color: #ddd; }
  1. Ejemplo de distribución de elementos en espacios iguales utilizando flexbox:
 
<div class="container"> <div class="item">Elemento 1</div> <div class="item">Elemento 2</div> <div class="item">Elemento 3</div> </div>
 
.container { display: flex; justify-content: space-between; } .item { margin: 10px; padding: 10px; background-color: #ddd; }
  1. Ejemplo de cambio de orden de elementos utilizando flexbox:
 
<div class="container"> <div class="item item-1">Elemento 1</div> <div class="item item-2">Elemento 2</div> <div class="item item-3">Elemento 3</div> </div>
 
.container { display: flex; } .item-1 { order: 3; } .item-2 { order: 2; } .item-3 { order: 1; } .item { margin: 10px; padding: 10px; background-color: #ddd; }
