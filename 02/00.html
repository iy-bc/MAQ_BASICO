Fundamentos en CSS

En esta lección aprenderás los conceptos básicos de programación con JavaScript, como variables, operadores y condicionales, así como la manipulación del contenido HTML y la interactividad con eventos. Aprenderás a seleccionar elementos HTML con el DOM y a aplicar estilos con CSS para mejorar la apariencia de tu sitio web. Al final de esta lección, podrás crear páginas web más dinámicas y atractivas.

240 min.
