Las media queries son una técnica de CSS que nos permite aplicar estilos específicos a un documento HTML según el tamaño de la pantalla o el dispositivo en el que se está visualizando. Esto es especialmente útil para adaptar el diseño y la funcionalidad de un sitio web a distintos dispositivos, como móviles, tabletas y ordenadores de escritorio.

La sintaxis básica de una media query es la siguiente:

 
@media (condición)
{
/* Estilos específicos para la condición */
}

La condición puede ser una expresión que evalúa el ancho, la altura, la orientación, la relación de aspecto, la densidad de píxeles o el modo de color de la pantalla. Algunos ejemplos de condiciones comunes son:

  • max-width: establece un ancho máximo para la pantalla.
  • min-width: establece un ancho mínimo para la pantalla.
  • orientation: establece la orientación de la pantalla en modo horizontal o vertical.
  • min-resolution: establece una resolución mínima para la pantalla.
  • max-device-width: establece un ancho máximo para el dispositivo.

A continuación, se presentan algunos ejemplos de media queries para diferentes tamaños de pantalla:

 
/* Estilos para pantallas con un ancho máximo de 600 píxeles */
@media (max-width: 600px)
{
body
{ font-size: 16px; }
}
 
/* Estilos para pantallas con un ancho mínimo de 800 píxeles */
@media (min-width: 800px) { body { font-size: 20px; } }
/* Estilos para pantallas en modo horizontal */
@media (orientation: landscape)
{
body
{ background-color: blue; }
}

También es posible combinar varias condiciones en una sola media query, por ejemplo:

 
/* Estilos para pantallas con un ancho máximo de 600 píxeles en modo vertical */
@media (max-width: 600px) and (orientation: portrait)
{
body
{ font-size: 16px; }
}

Es importante tener en cuenta que el uso de media queries no es una solución definitiva para el diseño responsive, ya que puede haber una gran variedad de tamaños y resoluciones de pantalla. Sin embargo, es una herramienta útil para proporcionar una experiencia de usuario consistente y optimizada en diferentes dispositivos.
