Propiedades CSS como color, fondo, fuente, tamaño y borde son muy importantes para el diseño y la apariencia de una página web. Aquí te explicamos su sintaxis, los colores y las unidades que se utilizan en ellas:

  • color: Esta propiedad establece el color del texto. La sintaxis es la siguiente: color: valor;. El valor puede ser un nombre de color (por ejemplo, "rojo" o "azul"), un código hexadecimal (por ejemplo, "#FF0000" o "#0000FF"), un RGB o RGBA (por ejemplo, "rgb(255,0,0)" o "rgba(0,0,255,0.5)").

  • fondo: Esta propiedad establece el color o imagen de fondo de un elemento. La sintaxis es la siguiente: background: valor;. El valor puede ser un nombre de color, un código hexadecimal, un RGB o RGBA, o una imagen de fondo (por ejemplo, "url('imagen.jpg')"). Además, esta propiedad tiene sub-propiedades como background-color, background-image, background-repeat, background-position y background-size para una mayor personalización.

  • fuente: Esta propiedad establece la familia de fuentes y su tamaño. La sintaxis es la siguiente: font: tamaño/line-height familia;. El valor del tamaño puede ser un número seguido de una unidad de medida (por ejemplo, "16px" o "1em"), mientras que la familia puede ser una lista de nombres de fuentes separados por comas (por ejemplo, "Arial, sans-serif"). Además, esta propiedad tiene sub-propiedades como font-size, font-family, font-style, font-weight y line-height para una mayor personalización.

  • tamaño: Esta propiedad establece el tamaño de un elemento. La sintaxis es la siguiente: width: valor; height: valor;. El valor puede ser un número seguido de una unidad de medida (por ejemplo, "100px" o "50%").

  • borde: Esta propiedad establece el borde de un elemento. La sintaxis es la siguiente: border: valor;. El valor puede ser un número seguido de una unidad de medida (por ejemplo, "1px"), seguido de un estilo de borde (por ejemplo, "solid" o "dotted"), seguido de un color (por ejemplo, "rojo" o "#FF0000"). Además, esta propiedad tiene sub-propiedades como border-width, border-style, border-color, border-radius y border-image para una mayor personalización.

Es importante mencionar que existen muchas otras propiedades CSS que pueden ser utilizadas para personalizar el diseño de una página web. Cada una de ellas tiene su propia sintaxis, colores y unidades utilizadas.

Unidades de CSS

Unidad Símbolo Explicación
Píxel px Una unidad de medida que representa un punto en la pantalla. Es una unidad absoluta y siempre tendrá el mismo tamaño independientemente del tamaño de la pantalla o del dispositivo.
Porcentaje % Una unidad de medida relativa que se basa en el tamaño del elemento padre. Por ejemplo, si un elemento tiene un ancho del 50% y su elemento padre tiene un ancho de 500 píxeles, el elemento tendrá un ancho de 250 píxeles.
Em em Una unidad de medida relativa que se basa en el tamaño de fuente del elemento padre. Por ejemplo, si un elemento tiene un tamaño de fuente de 1.5 em y su elemento padre tiene un tamaño de fuente de 16 píxeles, el tamaño de fuente del elemento será de 24 píxeles (1.5 * 16).
Rem rem Una unidad de medida relativa que se basa en el tamaño de fuente del elemento raíz del documento (normalmente el elemento html). A diferencia de em, rem siempre se basa en el mismo tamaño de fuente independientemente del elemento padre.
Viewport Width vw Una unidad de medida relativa que se basa en el ancho de la ventana gráfica (viewport) del dispositivo. Por ejemplo, si un elemento tiene un ancho del 50vw, tendrá un ancho igual al 50% del ancho de la ventana gráfica del dispositivo.
Viewport Height vh Una unidad de medida relativa que se basa en la altura de la ventana gráfica (viewport) del dispositivo. Por ejemplo, si un elemento tiene un alto del 50vh, tendrá un alto igual al 50% de la altura de la ventana gráfica del dispositivo.
Viewport Minimum vmin Una unidad de medida relativa que se basa en el tamaño más pequeño entre el ancho y la altura de la ventana gráfica del dispositivo. Por ejemplo, si el tamaño mínimo de la ventana gráfica es de 800 píxeles de ancho y 600 píxeles de alto, 1vmin equivaldrá a 6 píxeles (el 1% de 600 píxeles).
Viewport Maximum vmax Una unidad de medida relativa que se basa en el tamaño más grande entre el ancho y la altura de la ventana gráfica del dispositivo. Por ejemplo, si el tamaño máximo de la ventana gráfica es de 800 píxeles de ancho y 600 píxeles de alto, 1vmax equivaldrá a 8 píxeles (el 1% de 800 píxeles).

 

Además de las unidades más comunes como px, %, em, rem, vw, vh, existe una variedad de unidades menos utilizadas en CSS, a continuación se detallan algunas de ellas:

  • in: unidad de medida en pulgadas.
  • pt: unidad de medida en puntos, una medida de impresión común.
  • pc: unidad de medida en picas, una medida de impresión común.
  • cm: unidad de medida en centímetros.
  • mm: unidad de medida en milímetros.
  • ex: unidad de medida basada en la altura de la letra "x" en la fuente actual.
  • ch: unidad de medida basada en la anchura de la letra "0" en la fuente actual.
  • lh: unidad de medida basada en la altura de línea de la fuente actual.
  • fr: unidad de medida que divide el espacio disponible en partes iguales. Por ejemplo, si una fila tiene tres columnas, cada una con una anchura de 1fr, las columnas tendrán la misma anchura.

Es importante tener en cuenta que no todas las unidades son compatibles con todos los navegadores y que algunas unidades pueden tener un comportamiento diferente dependiendo del contexto en el que se utilicen.

Funciones CSS

Las funciones CSS son una serie de herramientas que permiten a los desarrolladores web realizar ciertas tareas específicas al aplicar estilos a una página web. Las funciones pueden ser utilizadas en valores de propiedades CSS para aplicar diferentes efectos y transformaciones en elementos HTML. Las funciones permiten hacer cálculos matemáticos, utilizar variables personalizadas, definir colores, transformaciones, entre otros.

Nombre de la función Código Explicación Ejemplo
URL url() Especifica la ubicación de un recurso externo, como una imagen o una fuente, que se utilizará en el documento. background-image: url("https://ejemplo.com/imagen.jpg");
RGBA rgba() Define un color utilizando los valores RGB y un valor de opacidad alpha. color: rgba(255, 0, 0, 0.5);
HSLA hsla() Define un color utilizando los valores HSL y un valor de opacidad alpha. background-color: hsla(120, 100%, 50%, 0.3);
GRADIENTE LINEAL linear-gradient() Crea un fondo que cambia gradualmente de un color a otro en una dirección específica. background: linear-gradient(red, yellow);
GRADIENTE RADIAL radial-gradient() Crea un fondo que cambia gradualmente de un color a otro desde un punto central hacia afuera en todas las direcciones. background: radial-gradient(circle, red, yellow);
ROTATAR rotate() Rota un elemento según un ángulo especificado. transform: rotate(45deg);
ESCALAR scale() Escala un elemento según los valores especificados para el eje X e Y. transform: scale(2, 0.5);
TRANSLADAR translate() Desplaza un elemento según los valores especificados para el eje X e Y. transform: translate(50px, -20px);
INCLINAR skew() Inclina un elemento según los valores especificados para el ángulo y el eje. transform: skew(30deg, 20deg);
CÍRCULO circle() Crea una forma de círculo. shape-outside: circle(50%);
ELIPSE ellipse() Crea una forma de elipse. shape-outside: ellipse(50% 50%);
REPETIR repeat() Especifica cómo se deben repetir las imágenes de fondo. background-repeat: repeat-x;
CUENTA count() Especifica cómo se deben contar las páginas cuando se imprime un documento. counter-reset: seccion;
CONTADOR counter() Crea un contador para un elemento específico. content: c
GRADOS deg(angle) Convierte un ángulo en grados a radianes. transform: rotate(deg(45));
ATRIBUTO attr(attribute) Obtiene el valor de un atributo HTML y lo utiliza en la declaración de estilo. content: attr(data-tooltip);
VARIABLE var(name, [fallback]) Utiliza una variable personalizada definida en el :root selector y proporciona un valor de reserva en caso de que la variable no esté definida. color: var(--primary-color, blue);
CALCULAR calc(expression) Realiza cálculos matemáticos en valores CSS. width: calc(100% - 20px);
