CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para aplicar estilos y dar formato a los elementos HTML en una página web. La sintaxis básica de CSS incluye dos partes principales: un selector y un bloque de declaración. El selector indica a qué elementos HTML se aplicará el estilo, mientras que el bloque de declaración contiene una o más propiedades CSS y sus valores asociados.

CSS se introdujo por primera vez en 1996 como una solución para separar la presentación del contenido en documentos web. Desde entonces, ha pasado por varias versiones importantes, cada una con nuevas características y mejoras:

  • CSS1: se publicó en 1996 y definió las especificaciones básicas de CSS, incluyendo la sintaxis de las reglas CSS y las propiedades de estilo.
  • CSS2: se publicó en 1998 y amplió la especificación para incluir soporte para hojas de estilo externas, mejoras en la capacidad de diseño de la página y nuevas propiedades de estilo.
  • CSS2.1: se publicó en 2004 y corrigió errores y ambigüedades en CSS2.
  • CSS3: es la versión actual de CSS y se compone de varios módulos que se están desarrollando de forma independiente, como Selectores, Texto, Fondos y bordes, Efectos y animaciones, Transformaciones y Diseño de la página. Cada módulo se desarrolla por separado y se publica cuando está listo, lo que permite a los desarrolladores utilizar las características de CSS3 en función de sus necesidades.

Formas de implentación de CSS

Hay tres formas de aplicar estilos CSS a una página web:

  1. Estilos en línea: se pueden aplicar estilos CSS directamente en un elemento HTML utilizando el atributo style. Por ejemplo:
 
<h1 style="color: red; font-size: 24px;">Título de la página</h1>

En este ejemplo, se está aplicando un estilo en línea al elemento h1, estableciendo el color del texto en rojo y el tamaño de fuente en 24 píxeles.

  1. Estilos en la sección head: se pueden aplicar estilos CSS en la sección head de una página HTML utilizando la etiqueta style. Por ejemplo:
 
<head> <style> h1 { color: red; font-size: 24px; } </style> </head>

En este ejemplo, se está aplicando un estilo al elemento h1 de la página, estableciendo el color del texto en rojo y el tamaño de fuente en 24 píxeles.

  1. Estilos en un archivo externo: se pueden aplicar estilos CSS en un archivo externo y luego vincular ese archivo a una página HTML utilizando la etiqueta link. Por ejemplo:
 
<head> <link rel="stylesheet" type="text/css" href="estilos.css"> </head>

En este ejemplo, se está vinculando un archivo externo llamado estilos.css que contiene estilos CSS para la página web.

Aquí hay un ejemplo completo de un archivo HTML que utiliza estilos CSS:

 
<!DOCTYPE html> <html> <head> <link rel="stylesheet" type="text/css" href="estilos.css"> </head> <body> <h1>Título de la página</h1> <p>Este es un párrafo de ejemplo.</p> <ul> <li>Elemento de lista 1</li> <li>Elemento de lista 2</li> <li>Elemento de lista 3</li> </ul> </body> </html>

Y aquí está el contenido del archivo estilos.css vinculado en el ejemplo anterior:

 
h1 { color: red; font-size: 24px; } p { color: blue; font-size: 18px; } ul { list-style-type: circle; }

En este ejemplo, se están aplicando estilos a los elementos h1, p y ul de la página, estableciendo el color y el tamaño de fuente del texto, así como el tipo de viñeta utilizado en la lista.
