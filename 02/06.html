CSS (Cascading Style Sheets) es un lenguaje de estilo que se utiliza para definir la presentación y el diseño de una página web. CSS se utiliza para separar el contenido HTML de la presentación visual y permite crear diseños más complejos y sofisticados.

Resumen de propiedades

  • Color: Define el color del texto y del fondo.
  • Fondo: Define el fondo de un elemento.
  • Fuente: Define la fuente de texto que se va a utilizar.
  • Tamaño: Define el tamaño del texto.
  • Borde: Define el borde de un elemento.
  • Margen: Define el espacio alrededor de un elemento.
  • Padding: Define el espacio dentro del borde de un elemento.
  • Display: Define el modo de visualización de un elemento.
  • Flexbox: Define un modelo de diseño flexible y adaptable.
  • Media queries: Permite definir diferentes estilos para diferentes dispositivos y tamaños de pantalla.

Media queries

Las media queries son una característica de CSS que permite adaptar el diseño de una página web a diferentes dispositivos y tamaños de pantalla. Se utilizan para establecer diferentes estilos para diferentes resoluciones de pantalla, anchos y orientaciones. Por ejemplo, se pueden definir estilos específicos para pantallas pequeñas de teléfonos móviles o para pantallas grandes de ordenadores de escritorio. Las media queries se utilizan junto con CSS y se definen utilizando una sintaxis específica.

Ejemplo de CSS

Este es un ejemplo que reúne lo aprendido:

/* Estilo para todos los párrafos del documento */ p { font-size: 16px; /* Tamaño de fuente */ color: #333; /* Color de texto */ line-height: 1.5; /* Espacio entre líneas */ } /* Estilo para el encabezado del documento */ header { background-color: #f2f2f2; /* Color de fondo */ padding: 20px; /* Espacio interno */ text-align: center; /* Alineación de texto */ } /* Estilo para un botón específico */ #mi-boton { background-color: #007bff; /* Color de fondo */ color: #fff; /* Color de texto */ border: none; /* Sin borde */ padding: 10px 20px; /* Espacio interno */ border-radius: 4px; /* Bordes redondeados */ cursor: pointer; /* Cambia el cursor al pasar el ratón */ } /* Estilo para un enlace específico */ .mi-enlace { color: #007bff; /* Color de texto */ text-decoration: none; /* Sin subrayado */ } /* Media Query para dispositivos móviles */ @media only screen and (max-width: 768px) { /* Estilo para el encabezado en dispositivos móviles */ header { font-size: 20px; /* Tamaño de fuente */ padding: 10px; /* Espacio interno */ } /* Estilo para el botón en dispositivos móviles */ #mi-boton { padding: 8px 16px; /* Espacio interno */ font-size: 14px; /* Tamaño de fuente */ } }
