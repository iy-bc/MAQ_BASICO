Para aplicar estilos a elementos HTML5, podemos utilizar selectores de elementos, selectores de clase y selectores de ID en CSS.

Los selectores de elementos se refieren a un tipo de elemento HTML5 específico, como "p", "div", "h1", etc. Por ejemplo, para aplicar un estilo a todos los elementos "p" en un documento HTML5, podemos usar el selector de elementos de CSS como sigue:

 
p { color: red; }

Los selectores de clase nos permiten aplicar estilos a elementos HTML5 con una clase específica. Para asignar una clase a un elemento HTML5, podemos usar el atributo "class" y definir el nombre de la clase. Por ejemplo, podemos definir una clase "destacado" y aplicar estilos a todos los elementos que tengan esa clase como sigue:

 
.destacado { font-weight: bold; color: blue; }

Para aplicar esta clase a un elemento HTML5, podemos agregar el atributo "class" con el valor "destacado", por ejemplo:

 
<p class="destacado">Este texto está en negrita y azul</p>

Los selectores de ID se utilizan para aplicar estilos a elementos HTML5 con un ID específico. Para asignar un ID a un elemento HTML5, podemos usar el atributo "id" y definir un nombre de ID único. Por ejemplo, podemos definir un ID "encabezado" y aplicar estilos a ese elemento como sigue:

 
#encabezado { background-color: gray; color: white; }

Para aplicar este ID a un elemento HTML5, podemos agregar el atributo "id" con el valor "encabezado", por ejemplo:

 
<header id="encabezado"> <h1>Mi sitio web</h1> </header>

Es importante tener en cuenta que cada ID debe ser único en un documento HTML5 y no puede ser utilizado en más de un elemento. Además, los selectores de ID tienen una prioridad más alta que los selectores de clase y los selectores de elementos en CSS.

Reglas de prioridad CSS

La regla de orden de selectores CSS establece que el estilo de un elemento se aplica siguiendo el orden en que aparecen los selectores en la regla CSS. Por lo tanto, si hay dos reglas que se aplican al mismo elemento HTML, la regla que aparece en último lugar tendrá prioridad y se aplicará el estilo correspondiente. En cuanto a la prioridad de una propiedad CSS en diferentes casos, se aplica la siguiente jerarquía: Regla inline: La propiedad definida en el atributo style de un elemento HTML tiene la máxima prioridad y se aplicará sobre cualquier otra definición de la misma propiedad. ID: Si hay una regla que selecciona un elemento por su ID (usando el selector #), tendrá prioridad sobre cualquier otra regla que seleccione el mismo elemento por una clase, etiqueta o atributo. Clase o atributo: Si hay varias reglas que seleccionan el mismo elemento por una clase o atributo, se aplicará la regla que aparece en último lugar. Etiqueta: Si hay varias reglas que seleccionan el mismo elemento por su etiqueta, se aplicará la regla que aparece en último lugar. Es importante tener en cuenta que la prioridad de una propiedad CSS también puede verse afectada por la especificidad de los selectores utilizados. En general, se recomienda utilizar selectores específicos y evitar el uso de reglas inline o estilos en línea para facilitar el mantenimiento del código.

Selectores avanzados

Selector de etiqueta tag Selecciona todos los elementos que coinciden con la etiqueta especificada
Selector de clase .class Selecciona todos los elementos que tienen la clase especificada
Selector de ID #id Selecciona el elemento con el ID especificado
Selector de atributo [attr] Selecciona todos los elementos que tienen el atributo especificado
Selector de atributo y valor [attr="value"] Selecciona todos los elementos que tienen el atributo especificado con el valor especificado
Selector de atributo y valor que comienza con [attr^="value"] Selecciona todos los elementos que tienen el atributo especificado con un valor que comienza con el valor especificado
Selector de atributo y valor que termina con [attr$="value"] Selecciona todos los elementos que tienen el atributo especificado con un valor que termina con el valor especificado
Selector de atributo y valor que contiene [attr*="value"] Selecciona todos los elementos que tienen el atributo especificado con un valor que contiene el valor especificado
Selector descendiente a b Selecciona todos los elementos b que son descendientes de a
Selector de hermano adyacente a + b Selecciona el primer elemento b que sigue inmediatamente después del elemento a
Selector de hermanos .a ~ .b Selecciona todos los elementos b que son hermanos de .a y aparecen después de ella
Selector universal * Selecciona todos los elementos
:active Selecciona el elemento cuando está activo
:checked Selecciona los elementos de entrada marcados
:disabled Selecciona los elementos de entrada desactivados
:empty Selecciona los elementos vacíos
:enabled Selecciona los elementos de entrada habilitados
:first Selecciona el primer elemento de su tipo
:first-child Selecciona el primer elemento hijo
:focus Selecciona el elemento cuando tiene el foco
:hover Selecciona el elemento cuando el cursor está encima
:last-child Selecciona el último elemento hijo
:link Selecciona los enlaces no visitados
:not Selecciona elementos que no cumplen con un criterio
:nth-child Selecciona elementos según su posición en relación a otros hijos
:visited Selecciona los enlaces visitados

Estos pseudo-selectores son útiles para aplicar estilos a elementos específicos en función de su estado o posición en la estructura del documento HTML. Se utilizan añadiendo el selector a la regla de estilo, por ejemplo:

css
a:hover { color: red; } input:disabled { opacity: 0.5; }

En este ejemplo, el primer selector aplica un estilo de color rojo al enlace cuando el cursor se sitúa sobre él, mientras que el segundo selector reduce la opacidad de los elementos de entrada desactivados.
